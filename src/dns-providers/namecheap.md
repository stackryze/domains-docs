# Namecheap FreeDNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

Namecheap FreeDNS offers free DNS hosting with an easy-to-use interface, ideal for Namecheap customers.

## Quick Setup

1. Create a Namecheap account
2. Access FreeDNS service
3. Add your subdomain
4. Note your Namecheap nameservers
5. Register with indevs using your nameservers
