# deSEC DNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

deSEC is a free, privacy-focused DNS service with built-in DNSSEC support, perfect for security-conscious users.

## Quick Setup

1. Create a deSEC account at [desec.io](https://desec.io)
2. Create a domain for your subdomain
3. Note your deSEC nameservers
4. Register with indevs using your nameservers
