# ClouDNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

ClouDNS offers free DNS hosting with support for multiple record types and API access.

## Quick Setup

1. Create a ClouDNS account at [cloudns.net](https://www.cloudns.net)
2. Add your subdomain as a zone
3. Note your ClouDNS nameservers
4. Register with indevs using your nameservers
