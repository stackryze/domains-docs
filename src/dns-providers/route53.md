# AWS Route 53 Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

AWS Route 53 is Amazon's highly available and scalable DNS service, perfect for users already in the AWS ecosystem.

## Quick Setup

1. Create an AWS account
2. Create a hosted zone for your subdomain
3. Note your Route 53 nameservers
4. Register with indevs using your nameservers
