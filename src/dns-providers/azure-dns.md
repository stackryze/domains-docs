# Azure DNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

Azure DNS is Microsoft's DNS hosting service, providing high availability and integration with Azure services.

## Quick Setup

1. Create a Microsoft Azure account
2. Create a DNS zone for your subdomain
3. Note your Azure DNS nameservers
4. Register with indevs using your nameservers
