# Google Cloud DNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

Google Cloud DNS is a scalable, reliable DNS service running on Google's infrastructure, perfect for GCP users.

## Quick Setup

1. Create a Google Cloud account
2. Enable Cloud DNS API
3. Create a managed zone for your subdomain
4. Note your Cloud DNS nameservers
5. Register with indevs using your nameservers
