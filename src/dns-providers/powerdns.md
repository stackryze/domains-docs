# PowerDNS Self-Hosted Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

PowerDNS is a powerful, open-source DNS server software that you can host yourself for complete control over your DNS infrastructure.

## Quick Overview

1. Install PowerDNS on your server
2. Configure PowerDNS with your subdomain zone
3. Set up authoritative nameservers
4. Register with indevs using your nameservers

For detailed instructions, please check back soon or see the [Custom DNS Guide](custom) for general self-hosted DNS setup.
