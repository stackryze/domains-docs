# Hurricane Electric (HE.net) DNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

Hurricane Electric provides completely free DNS hosting with IPv6 support and DNSSEC capabilities.

## Quick Setup

1. Create a Hurricane Electric account at [dns.he.net](https://dns.he.net)
2. Add a new domain for your subdomain
3. Note your HE.net nameservers
4. Register with indevs using your nameservers
