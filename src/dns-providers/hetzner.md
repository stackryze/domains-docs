# Hetzner DNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

Hetzner DNS is a free DNS service provided by Hetzner, offering reliable DNS hosting with a simple API.

## Quick Setup

1. Create a Hetzner account
2. Access Hetzner DNS Console
3. Create a zone for your subdomain
4. Note your Hetzner nameservers
5. Register with indevs using your nameservers
