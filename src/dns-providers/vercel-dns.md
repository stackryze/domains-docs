# Vercel DNS Setup

> [!WARNING]
> These docs are in **Beta** and actively updated daily. Please report if you find any issues.

*Documentation coming soon*

Vercel DNS provides automatic SSL and edge network capabilities, perfect for frontend developers using Vercel.

## Quick Setup

1. Create a Vercel account
2. Add your subdomain to a Vercel project
3. Note your Vercel nameservers
4. Register with indevs using your nameservers
